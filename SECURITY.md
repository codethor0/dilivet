# Security Policy

If you believe you've found a security vulnerability, please do **not** open a public issue.

- Create a **private security advisory** on GitHub (Security → Advisories), or
- Email: <codethor@gmail.com>

We aim to acknowledge reports within 3 business days.

Supported versions: the latest released `vX.Y.Z` tag.
