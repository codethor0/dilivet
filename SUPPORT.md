# Support

Thanks for using DiliVet! Here's how to get help:

## Questions & Discussions

- **General questions:** Open a [GitHub Discussion](https://github.com/codethor0/dilivet/discussions)
- **Usage help:** Check the [README](./README.md) quick start and `dilivet -version`
- **Bug reports:** Open an [Issue](https://github.com/codethor0/dilivet/issues/new?template=bug_report.yml)

## Reporting Bugs

Use the **Bug Report** template and include:
- Go version (`go version`)
- OS/arch (`uname -sm`)
- Steps to reproduce
- Expected vs actual behavior

## Security Vulnerabilities

**Do not open public issues for security bugs.**  
See [SECURITY.md](./SECURITY.md) for responsible disclosure.
